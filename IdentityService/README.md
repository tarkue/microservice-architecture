# Identity Service

Микросервис, отвечающий за идентификацию пользователя в системе. Реализует методы авторизации, регистрации, получения информации о пользователе и добавления прав пользователю внутри системы.

## Развёртывание

Для развёртывания используется система контейнеризации в Docker, на данный момент необходимо добавить файл `.env` для работы проекта при запуске в docker compose

пример `.env` лежит в .env.example.

После необходимо запустить docker-контейнеры командой или внутри Rider

```bash
docker compose -f compose.yaml -p identityservice up -d
```
